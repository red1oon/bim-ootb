Out Of The Box
Version 1.0 (April – May 2026)
Frictionless BIM. Two DBs. One browser. Zero install.
Probably the lightest BIM app ever made.
Three.jsr160 ESM (BatchedMesh, DLOD)
SQLite3.44.2 (sql.js 1.10.3 WASM)
web-ifc0.0.77 (IFC2x3 + IFC4)
SheetJS0.20.3
Browser Compatibility
Chrome 90+ • Firefox 90+ • Safari 15+ • Edge 90+
Mobile: iOS Safari 15+ • Chrome Android
Requires: WebAssembly, WebGL 2, Web Workers
Architecture
80+ JS modules, vanilla ES6. No server, no APIs.
IFC parsed client-side. SQLite DBs ARE the app.
30 buildings. 18 languages. 17 country rates.
Features
3D/2D viewer • IFC import/export • Clash detection
4D time machine • 5D cost estimation • BOM engine
Grid system • ERP • PWA offline • City mode (786 buildings)

DIY Downloader
Download everything and run on your own machine!
WHAT YOU GET:
✓ Complete BIM viewer (~49MB download, ~4MB viewer)
✓ Libraries loaded from CDN on first use, then cached
✓ Drop IFC / OBJ / DAE / GLB files to view
✓ BOQ, 2D Plans, Section Cut, Walk Mode
✓ Your own domain, your own branding
✓ Still loads cloud buildings via URL if needed
THE SCRIPT WILL:
1. Check your system (Windows / Mac / Linux)
2. Install Python 3 if not found
3. Download BIM OOTB viewer + all libraries
4. Start a local server on port 8080
5. Open your browser — ready to use!
